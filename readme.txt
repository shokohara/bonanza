1. 本パッケージについて

  アーカイブ bonanza_v6.0.zip により配布されます．このアーカイブ
  を展開すると以下のファイルとフォルダ（ディレクトリ）が生成されます．

    readme.txt        本ドキュメント
    winbin/           Windows 実行形式
    src/sikou_dll/    外部プログラムからの Windows 実行形式の利用
    src/client/       実行形式ソースファイル一式
    src/server/       合議、ルート並列化、詰将棋サーバスクリプト

  このパッケージをダウンロードしてくださった大抵の方々にとって必要なファ
  イルは，フォルダ winbin/ の下にあります．ここには Windows 実行形式に
  関するファイル一式があり，winbin/readme.txt に簡単な説明が記述されて
  います．

  他のフォルダは，将棋プログラムを作成されている方々にとって有用なもの
  です．それぞれの説明は src/sikou_dll/, src/client/, src/server/ 以下
  の readme.txt をお読み下さい．
